You are an expert Java/Kotlin developer specializing in Minecraft client development. Your task is to build a fully custom, feature-rich Minecraft client launcher and mod framework from scratch — similar to Lunar Client but remade with additional unique features. The client must support Windows and Linux.

PROJECT OVERVIEW
Build a custom Minecraft client platform called [CLIENT_NAME] consisting of:
	1.	A native cross-platform Launcher (Windows + Linux)
	2.	A Core Mod/Agent that injects into Minecraft at runtime
	3.	A HUD & UI Framework rendered on top of Minecraft
	4.	A Settings & Profile System
	5.	An optional Backend API for cosmetics, stats, and authentication

TECH STACK
Launcher
	∙	Language: Java 17+ with JavaFX OR Electron + React + TypeScript (prefer Electron for modern UI)
	∙	Packaging: Use electron-builder for Windows (.exe installer) and Linux (.AppImage / .deb)
	∙	Auto-updater: Integrate electron-updater
Core Client Mod / Agent
	∙	Language: Java 17 (compiled to a fat JAR)
	∙	Injection method: Java Agent (-javaagent:) + Mixin framework (Fabric Mixin or SpongePowered Mixin)
	∙	Build tool: Gradle with shadow JAR plugin
	∙	Target Minecraft versions: 1.8.9 (for PvP), 1.12.2, 1.16.5, 1.18.2, 1.19.4, 1.20.x
	∙	Base: Build on top of Fabric Loader or use Forge as the mod loader backbone
Rendering / HUD
	∙	Use LWJGL (already bundled in Minecraft) for OpenGL rendering
	∙	Custom GUI framework built with scissor regions, layered rendering, and animation easing
	∙	Font rendering via Minecraft’s font renderer extended with custom TTF support
Settings Backend (optional but recommended)
	∙	Node.js + Express or FastAPI (Python)
	∙	Database: PostgreSQL for user data, Redis for sessions/caching
	∙	Auth: JWT tokens + optional Microsoft OAuth passthrough
